# Web Workbench

A collection of small browser-based utilities. Plain static HTML/CSS/JS — no framework, no build step.

Deployed via GitHub Pages.

## Tools

| Tool | Path | Description |
|------|------|-------------|
| Minutes | `/minutes/` | Elapsed time counter showing whole minutes |

## Adding a new tool

1. Create a new directory: `my-tool/`
2. Add an `index.html` inside it (self-contained or linking to shared assets)
3. Add a link card to the root `index.html`
4. Push to `main` — GitHub Pages deploys automatically
